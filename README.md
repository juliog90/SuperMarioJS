## How to run
You need NPM that is included with Node.JS

### Download and install if you don't have it
https://nodejs.org/en/ 

### Run the Server
Get this repository code and run inside the folder

```
npm start or
npm run start

```

### Load the game on chrome

Go to this direction htttp://localhost:5000
